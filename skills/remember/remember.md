# Remember Important Information

## Description
Explicitly store important information in memory for future reference. This is useful for remembering user preferences, important dates, names, or any information that should be recalled later.

## Usage
When the user shares important information they want remembered, use this skill to create a permanent note in their memory.

To use: "Remember: [important information]"

## Examples
- "Remember: User's birthday is March 15th"
- "Remember: User prefers Python over JavaScript"
- "Remember: User is working on a project called Atlas"
- "Remember: User's favorite color is blue"
