# Text to Speech

## Description
Speak a reply back to the user as a real Telegram voice message, using a local
TTS engine, ElevenLabs, or OpenAI. Only works from the Telegram bot - it has no
effect in the web dashboard chat, since there's no chat to send a voice message
into there.

## Usage
Use this skill when the user explicitly asks you to speak, say something out
loud, send a voice message, or read something back to them out loud.

## Examples
- Can you say that as a voice message?
- Read me the weather out loud
- Speak this using ElevenLabs
- Say "happy birthday" out loud
