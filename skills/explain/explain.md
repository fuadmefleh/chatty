# Explain Concept

## Description
Provide clear, detailed explanations of complex concepts, topics, or ideas. Can break down difficult subjects into easy-to-understand parts with examples and analogies.

## Usage
When a user needs help understanding a concept, use this skill to provide a comprehensive explanation.

To use: "Explain [concept/topic]"

## Examples
- "Explain how neural networks work"
- "Explain the difference between async and sync programming"
- "Explain quantum computing to a beginner"
- "Explain the ReACT pattern in AI agents"
