# Web Search

## Description
Search the web for current information, news, facts, or any topic that requires up-to-date knowledge beyond the AI's training data.

## Usage
When you need to find current information or verify facts, use this skill. Simply state what you want to search for.

To use: "Search the web for [topic]"

## Examples
- "Search the web for latest news about AI developments"
- "Search the web for weather in Tokyo"
- "Search the web for Python 3.12 release date"
