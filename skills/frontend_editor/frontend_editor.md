# Frontend Editor

## Description
Tools for reading, writing, and rebuilding the Chatty Vite/React frontend application.
The agent can use these tools to modify the dashboard UI (pages, components, styles,
routing) and rebuild so changes take effect.

## Usage
Use these tools when the user asks to change the web dashboard UI — new pages,
modified components, style updates, route changes, or API integration fixes.
Always rebuild after making source changes so the running frontend picks them up.

## Examples
- "Add a new Settings page to the dashboard"
- "Make the chat page show timestamps"
- "Update the sidebar to include a link to exercise tracking"
- "Fix the Notes page so it shows deletion confirmation"
