# Code Review

## Description
Review code for bugs, best practices, performance issues, security vulnerabilities, and suggest improvements. Can analyze any programming language.

## Usage
When a user shares code and wants feedback, use this skill to provide a thorough review.

To use: "Review this code: [code snippet]"

## Examples
- "Review this Python function for potential issues"
- "Review this code for security vulnerabilities"
- "Review this code and suggest performance improvements"
- "Review this code for Python best practices"
